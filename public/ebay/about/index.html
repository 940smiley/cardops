About CardOps AI

CardOps AI is a local-first sports and trading card operations project for inventory, image review, listing preparation, and eBay-safe workflows.

Purpose

This GitHub Pages site supports the CardOps AI eBay OAuth flow. The production callback URL is https://940smiley.github.io/cardops/ebay/callback/.

How The Callback Works

  • eBay redirects the browser to the callback URL after authorization or decline.
  • The static callback page returns HTTP 200 and attempts to forward the response to local CardOps.
  • CardOps runs locally at 127.0.0.1 and records only redacted OAuth evidence.

Relationship To eBay

CardOps AI is an independent project and is not owned, operated, sponsored, or endorsed by eBay.
