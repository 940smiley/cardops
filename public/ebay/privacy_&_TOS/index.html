Privacy

This GitHub Pages site hosts static support pages for CardOps AI and the eBay OAuth callback. It does not use cookies, analytics scripts, advertising scripts, or local browser storage.

During the eBay OAuth flow, eBay may redirect your browser to https://940smiley.github.io/cardops/ebay/callback/ with authorization parameters in the URL. The static callback page attempts to forward those parameters to the local CardOps API running on your machine at 127.0.0.1:8000.

  • The static site does not intentionally store OAuth authorization codes.
  • CardOps local logs should record only redacted OAuth evidence.
  • GitHub may process standard server logs because this site is hosted on GitHub Pages.
